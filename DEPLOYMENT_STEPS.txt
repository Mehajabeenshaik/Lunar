  → Time: 5 min

Step 2: Clone Repository  
  → mkdir C:\HF_Deployment && cd C:\HF_Deployment
  → git clone https://huggingface.co/spaces/<YOUR_USERNAME>/warehouse_env
  → cd warehouse_env
  → Time: 5 min

Step 3: Copy Project Files
  → Copy-Item "C:\Users\HP\Documents\lunar\warehouse_env\*" ".\" -Recurse -Force
  → Time: 5 min

Step 4: Commit to Git
  → git add .
  → git commit -m "Initial warehouse environment deployment"
  → Time: 5 min

Step 5: Push to HuggingFace
  → git push
  → (Enter your HF credentials when prompted)
  → Time: 3 min

Step 6: Add Secrets
  → Go to Space Settings → Repository secrets
  → Add OPENAI_API_KEY or HF_TOKEN
  → Time: 5 min

Step 7: Monitor Build
  → Watch the build logs (10-15 minutes)
  → Space will show "Running" when complete
  → Time: 15 min

Step 8: Test Endpoints
  → curl https://<your-space>/health
  → Should return {"status":"ok"}
  → Time: 5 min

Step 9: Final Validation
  → Run validation script (if available)
  → Or test manually with Swagger UI
  → Time: 5 min

Total Time: ~60 minutes
